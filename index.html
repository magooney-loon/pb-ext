Core Features

System Monitoring

Real-time metrics for CPU, memory, disk, network, and runtime stats. Monitor your server's health at a glance.

API Schema

Automatically generates openapi like docs and schemas on runtime discovery.

Structured Logging

Comprehensive logging with error tracking and request tracing for better debugging and system insights.

Visitor Analytics

Track and analyze visitor statistics, page views, device types, and browsers to understand your user base.

PocketBase Integration

Seamlessly piggybacks off PocketBase's superuser authentication and CSS for a consistent experience.

Architecture

pb-ext follows a modular architecture with clean separation of concerns:

cmd/server/

Server initialization and startup logic

core/logging/

Structured logging and error handling

core/monitoring/

System metrics collection and health monitoring

core/server/

Core server implementation and web dashboard

For detailed architecture information and implementation details, visit our GitHub documentation.
